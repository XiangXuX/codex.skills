# Budget Rules

## Default Budgeting

- Default people count: 2 adults unless specified otherwise
- Show local currency first
- Add CNY beside local currency only when useful to the user
- Mark all rough values as estimated

## Include

- Long-distance transport
- Local transport or car rental
- Lodging
- Fuel when self-driving
- Core passes / entry fees

## Exclude By Default

- Shopping
- Travel insurance
- Optional premium dining
- Personal miscellaneous spending

## Reporting Style

- Prefer ranges over false precision
- Say what assumptions drive the range
- Separate required costs from optional add-ons

# Questioning Framework

Ask only the next highest-value questions. Prefer 1-3 concise questions at a time.

## Priority Order

1. Core shape
- From where?
- When?
- How many days or nights?
- How many people?

2. Planning constraints
- Budget sensitivity
- Preferred transport
- Must-visit places
- Places or activities to avoid

3. Style choices
- Fast-paced or relaxed
- Nature / city / food / culture / road trip balance
- Self-drive vs public transport vs tours

## Explore-Phase Prompt Pattern

Use a format like:

- "Based on your dates and pace, there are three good directions."
- "Option A is the smoothest."
- "Option B is cheaper but more effort."
- "Option C is more scenic but requires earlier starts."

Then ask the user to choose what to optimize:
- Lowest cost
- Smoothest logistics
- Most scenic / highest experience

## Converge-Phase Prompt Pattern

Summarize:
- Fixed: ...
- Preferred: ...
- Flexible: ...
- Excluded: ...

Then ask only unresolved, route-shaping questions.

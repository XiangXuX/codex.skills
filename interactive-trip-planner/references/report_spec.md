# Report Spec

Default report sections:

1. Recommendation Summary
- Best overall plan
- Why it wins

2. Planning Brief
- Origin
- Dates / duration
- People
- Transport style
- Must-haves
- Exclusions

3. Options Compared
- Transport options
- Routing variants
- Why some are not recommended

4. Lodging / Local Transport
- Hotels or areas
- Car / rail / ferry / tours as relevant

5. Final Itinerary
- Day-by-day structure

6. Budget
- Estimated totals
- Required vs optional

7. Backup Plan
- What to do if the first choice is unavailable

8. Sources
- Link list
- Price timing disclaimer

## Tone

- Clear
- Practical
- Decision-oriented
- Suitable to forward to someone else
